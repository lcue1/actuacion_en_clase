Selectores CSS

Selector Universal

Selecciona todos los elementos del documento sin importar la etiqueta.


* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
  font-family: Arial, sans-serif;
}

Selector de Etiqueta

Ejemplo para la etiqueta <p>.

Este párrafo usa el selector de etiqueta.


p {
  font-size: 1.2rem;
  color: #ff5722;
  line-height: 1.6;
}

Selector de Clase

Este div usa una clase llamada .selectorDeClase.

.selectorDeClase {
  background-color: #d0ebff;
  padding: 1rem;
  border-left: 5px solid #339af0;
  font-weight: bold;
}

Selector de ID

Este título usa un ID llamado #tituloPrincipal


#tituloPrincipal {
  text-align: center;
  font-size: 1.5rem;
  color: #2b8a3e;
}

Selector Descendente

Ejemplo: nav ul li a aplica a los enlaces del menú dentro de una lista.


nav ul li a {
  color: white;
  text-decoration: none;
  padding: 0.5rem 1rem;
}

Pseudoclase :hover

Pasa el mouse sobre los enlaces del menú para ver el efecto visual aplicado con :hover.


nav ul li a:hover {
  background-color: #d9480f;
  border-radius: 4px;
}
